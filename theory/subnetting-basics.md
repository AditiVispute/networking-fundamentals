# Subnetting Basics

## 📌 Overview

Subnetting is the process of dividing a large network into smaller sub-networks called subnets.

It helps in improving network performance and enhancing security.

---

## 🔹 Why Subnetting is Required?

- Reduces network congestion
- Improves network performance
- Enhances security
- Efficient use of IP addresses

---

## 🔹 What is a Subnet Mask?

A subnet mask is used to divide an IP address into:

- Network portion
- Host portion

Example:

IP Address: 192.168.1.10  
Subnet Mask: 255.255.255.0

---

## 🔹 Network Address

Represents the starting address of a subnet.

Example:

IP: 192.168.1.10  
Subnet Mask: 255.255.255.0  
Network Address: 192.168.1.0

---

## 🔹 Broadcast Address

Used to send data to all devices in a network.

Example:

Broadcast Address: 192.168.1.255

---

## 🔹 Benefits of Subnetting

- Efficient IP address management
- Improved network performance
- Better network organization
- Increased security

---

## 🎯 Key Takeaways

- Subnetting divides a network into smaller parts.
- Helps reduce traffic and improve performance.
- Uses subnet mask to identify network and host.
