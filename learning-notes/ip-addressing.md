# IP Addressing

## 📌 Overview

An IP Address (Internet Protocol Address) is a unique numerical label assigned to each device connected to a computer network.

It helps in identifying devices and enabling communication between them over a network.

---

## 🔹 Types of IP Address

### 1️⃣ IPv4

- 32-bit address
- Written in decimal format
- Example: 192.168.1.1
- Most commonly used IP version

---

### 2️⃣ IPv6

- 128-bit address
- Written in hexadecimal format
- Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
- Introduced due to shortage of IPv4 addresses

---

## 🔹 Public IP Address

- Assigned by Internet Service Provider (ISP)
- Used to identify devices over the internet

---

## 🔹 Private IP Address

- Used within local networks
- Cannot be accessed directly from the internet
- Example:
  - 192.168.x.x
  - 10.x.x.x
  - 172.16.x.x to 172.31.x.x

---

## 🔹 Static IP Address

- Fixed IP address
- Does not change
- Used for servers

---

## 🔹 Dynamic IP Address

- Automatically assigned
- Changes periodically
- Used for home networks

---

## 🎯 Key Takeaways

- IP Address uniquely identifies devices on a network.
- IPv4 is widely used, IPv6 is the future.
- Public IP is used for internet communication.
- Private IP is used within local networks.
