# TCP vs UDP

## 📌 Overview

TCP (Transmission Control Protocol) and UDP (User Datagram Protocol) are communication protocols used at the Transport Layer of the OSI Model.

They are responsible for transmitting data between devices over a network.

---

## 🔹 TCP (Transmission Control Protocol)

### Features:
- Connection-oriented
- Reliable data transfer
- Error checking and correction
- Data packets delivered in order

### Use Cases:
- Web browsing (HTTP/HTTPS)
- Email services
- File transfers (FTP)

---

## 🔹 UDP (User Datagram Protocol)

### Features:
- Connectionless
- Faster data transmission
- No error correction
- Data packets may arrive out of order

### Use Cases:
- Video streaming
- Online gaming
- Voice calls (VoIP)

---

## 📊 Difference Between TCP and UDP

| Feature        | TCP                  | UDP               |
|---------------|----------------------|-------------------|
| Connection    | Connection-oriented  | Connectionless    |
| Reliability   | Reliable             | Unreliable        |
| Speed         | Slower               | Faster            |
| Error Check   | Yes                  | No                |
| Packet Order  | Maintained           | Not maintained    |

---

## 🎯 Key Takeaways

- TCP ensures reliable communication.
- UDP provides faster data transmission.
- Choice of protocol depends on application requirements.
